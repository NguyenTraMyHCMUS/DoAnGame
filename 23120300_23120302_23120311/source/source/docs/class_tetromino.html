Detailed Description

Lớp cơ sở trừu tượng đại diện cho khối Tetris trong trò chơi.

Lớp này quản lý các thao tác như: xoay, di chuyển, vẽ, kiểm tra tính hợp lệ và lưu trữ trạng thái của khối.
