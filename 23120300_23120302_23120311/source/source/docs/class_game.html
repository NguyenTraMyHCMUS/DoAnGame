Detailed Description

Lớp chính điều khiển trò chơi Tetris: xử lý vòng lặp game, vẽ, quản lý trạng thái.

Chứa các đối tượng chính như Field, Tetromino, GameState, ScoreManager, LevelManager. Xử lý đầu vào, cập nhật trạng thái và vẽ màn hình.
