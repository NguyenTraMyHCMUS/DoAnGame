Detailed Description

Trạng thái hiển thị hướng dẫn của trò chơi.

Lớp này kế thừa từ GameState (interface) và chịu trách nhiệm hiển thị giao diện hướng dẫn chơi. Trạng thái này hiển thị nội dung hướng dẫn và quy tắc chơi của game.
