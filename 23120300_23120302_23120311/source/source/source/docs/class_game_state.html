Detailed Description

Lớp trừu tượng đại diện cho một trạng thái trong trò chơi.

Lớp trừu tượng này định nghĩa các phương thức cơ bản mà các trạng thái cụ thể (như MainMenuState, PlayingState, PauseState, GameOverState) cần phải triển khai.
