Detailed Description

Quản lý lưới trò chơi Tetris, bao gồm kiểm tra va chạm, xóa dòng và vẽ.

Lớp này chịu trách nhiệm giữ trạng thái các ô trong lưới, kiểm tra tính hợp lệ của vị trí, vẽ lưới, và xử lý logic xóa dòng.
