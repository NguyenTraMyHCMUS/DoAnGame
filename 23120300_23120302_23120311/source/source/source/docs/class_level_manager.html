Detailed Description

Quản lý cấp độ hiện tại và số dòng đã xóa trong trò chơi.

Lớp này theo dõi tiến trình của người chơi bằng cách cập nhật cấp độ dựa trên số dòng đã xóa. Việc tách riêng logic này giúp tuân thủ nguyên lý SRP (Single Responsibility Principle) trong SOLID.
